# Elastic Search

Elastic Search indexes used by UserOrg :&#x20;

* usernotes
* location
* userfeed
* user\_v3
* org\_v3

User\_v3 index is mapped to user\_alias and org\_v3 index mapped to org\_alias.&#x20;
